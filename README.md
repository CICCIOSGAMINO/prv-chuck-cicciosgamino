## Architecture 
The snap can be installed on both architecture **amd64** and **armhf** : 

```bash
# install the snap 
sudo snap install prv-chuck-cicciosgamino

# install the snap if in private mode 
snap login 
sudo snap install prv-chuck-cicciosgamino
```
